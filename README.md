# A voting app based on the TALL stack, taught by `Andre Madarang` on `Laracasts`

## About the project
Zabe is an idea voting app where users can create ideas and have other users vote on them. Some other features
- Comment on ideas
- Filter ideas
- and so on.

### Built with
- Tailwind CSS
- Alpine.js
- Laravel
- Livewire
- PHPUnit
- Composer
- MySQL
